RMS Current

Equation:

$$I_\text{rms} = \frac{I_0}{\sqrt{2}}$$

Variables:

  • \(I_\text{rms}\): Root mean square current [A]
  • \(I_0\): Peak current [A]
This equation provides the root mean square (RMS) value of the current in an AC circuit, which is the equivalent direct current (DC) value that delivers the same power.
